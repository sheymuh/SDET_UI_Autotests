# Test Suite: Проверка страницы авторизации

## TC-01. Авторизация пользователя

### Предусловия:
1. Открыть браузер
2. Перейти на сайт https://www.way2automation.com/angularjs-protractor/registeration/#/login

### Шаги:
1. Заполнить поля для авторизации: "Username", "Password", "Username *" (username description) (см. "Ожидаемый результат")
2. Нажать кнопку "Login"

## TC-02. Навигация назад в браузере после успешной авторизации

### Предусловия:
1. Открыть браузер
2. Перейти на сайт https://www.way2automation.com/angularjs-protractor/registeration/#/login

### Шаги:
1. Заполнить поля данными для успешной авторизации (см. TC-01, первая строка таблицы "Ожидаемый результат") 
2. Нажать кнопку "Login"
3. Нажать стрелочку назад в браузере

### Ожидаемый результат:
Все поля ввода пустые
